About the Project

cranstats was written as part of Google Summer of Code 2012 by Tim Jurka and Hadley Wickham.

Overview

The goal of this project is to collect package download data from CRAN mirrors in a central location. cranstats is hosted on Heroku and Amazon S3. The web interface is built on top of web.py and uses Twitter Boostrap, jQuery, and flot for the interface. The data is stored in a PostgreSQL database and results are cached using memcached.
